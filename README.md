# Invisibility-Cloak

A Programm that enabels you to see through a specificly chosen color.

how to operate :
After running the code immediatly  hide from your webcam for the first 5 seconds,
as it needs to capture the background (without you in it)
After the 5 seconds take  any object with the color you've chosen and show it to the camera, you should be able to see the background
through the object, if not try to turn off the bluetooth since it can activate your phone's camera and 
run the programm again .
